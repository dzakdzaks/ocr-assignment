# ocr-assignment

This is an assignment single module clean architecture project using MLKit Text Recognition, get current location, find path between 2 location, and upload the data to Firebase Firestore.

## Tech Stack
- [Kotlin](https://kotlinlang.org/)
- [Coroutine](https://kotlinlang.org/docs/coroutines-overview.html)
- [Flow](https://kotlinlang.org/docs/flow.html#flows)
- [StateFlow](https://kotlinlang.org/api/kotlinx.coroutines/kotlinx-coroutines-core/kotlinx.coroutines.flow/-state-flow/)
- [Retrofit](https://square.github.io/retrofit/)
- [Moshi](https://github.com/square/moshi)
- [Dagger-Hilt](https://dagger.dev/hilt/)
- [CameraX](https://developer.android.com/training/camerax)
- [Firebase Firestore](https://firebase.google.com/docs/firestore)
